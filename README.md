# Vamoss exp

### instalar

```
yarn
```

### rodar local

```
yarn electron:serve
```

##

O aplicativo abre duas janelas.
Uma com o editor de codigo e outra com o visualizador.

Arquivos salvos pelo editor vão para a pasta /public/main/files. Assim eles ficam disponíveis para o visualizador carregar as sketches.
